# 模板目录

将考勤表 Word 模板放在此目录，命名为 `考勤表模板.docx`。

首次使用可运行项目根目录下的 `python create_template.py` 生成通用模板。

## 模板结构要求

- 第一个表格：27 行 x 13 列
- 行 0-4：课程名称、教学内容、参训人数、培训课时、培训日期、教员姓名、培训时间、培训地点
- 行 5：填写说明
- 行 6：表头（序号、工号、姓名、签到）
- 行 7-26：学员数据区（左 20 人 + 右 20 人，共 40 个位置）

若使用自定义模板，可用 `inspect_template.py` 检查表格结构是否与 `generate_docx.py` 中的索引匹配。
